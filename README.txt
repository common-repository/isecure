== Description ==

= Why this plugin should get installed ? =

Wordpress plugin and theme is increaseing day by day. Many of the plungins features sounds so good that without careing about it's origin we installs it. Plugin developer may be learner, professional, hacker and spammer. 

Some plugins/themes makes your server slow and some do unusual activities on your server.

Example:
* Sending advertisement emails to unknown people without your notice. Which makes your site black listed.
* Some useing lot's of unwanted memory usages which makes your site slow. 
* Bad Memeber deletes post/page/plugin and stays quite. Now you can catch who did it. 

= Features =

* User Activity (Records log of his each activity. Eg. When and what he did.)
* Email Activity (Records all email sent from your server.)
* Number of post, page, comments at a glance at dashboard.
* Visitors Activity (Records every visitors IP, date, time, page name and many more.)

== Installation ==

= Automatic Installation =

* Go to your plugin browser inside your wordpress installation and search `isecure` by keyword. Then choose iSecure and click install. It will be installed shortly.
* Activate the plugin from `Plugins` menu after installation

= Manual Installation =

* Download the latest version and extract the folder to the `/wp-content/plugins/` directory
* The plugin will appear as inactive in your `Plugins` menu
* Activate the plugin through the `Plugins` menu in WordPress
